## 1. Playwright Screenshot Test

Add to `frontend/e2e/docs-screenshots.spec.ts` within the existing theme loop.

**`beforeAll` setup:** Spawn a PySCF script that:
1. Builds H2O (`gto.M(atom="O 0 0 0; H 0 0 0.96; H 0 0.96 0", basis="sto-3g")`)
2. Runs RHF, evaluates HOMO on a 40x40x40 grid
3. Stores cube data in `atoms.info["orbital_homo"]`
4. Creates two `Isosurface` geometries: positive lobe (blue `#2244CC`) and negative lobe (red `#CC4422`)
